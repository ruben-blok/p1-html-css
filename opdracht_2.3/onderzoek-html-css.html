Onderzoek HTML & CSS

1. Waarom noem je de eerste HTML-pagina meestal index.html?

De eerste HTML-pagina wordt meestal index.html genoemd omdat een webserver bij een bezoek aan een domeinnaam automatisch naar dit specifieke bestand zoekt als standaard homepage.

2. Met welke versie van HTML werken we op dit moment (2025)?

Html 5.

3. Met welke versie van CSS werken we op dit moment (2025)?

CSS 3.

4. Waarom ziet een website er soms net iets anders uit in verschillende browsers, zoals Chrome, Firefox, Edge of Safari?

Websites zien er in verschillende browsers verschillend uit vanwege de rendering engines van elke browser die de code van een website op unieke manieren interpreteren.

Welke drie webbrowsers zijn op dit moment het populairst? Noem ze in volgorde van gebruik/populariteit.

  1. Google Chrome
  2. Safari
  3. Firefox
